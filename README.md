## Manual-del-lenguaje-LCON ##

Problema:
Leer dos números y luego imprimir su suma

En el Basic antiguo:
a = input("Número a = ")
a = val(a)
b = val(input("Número b = "))
c = a + b
print(c)

En JavaScript actual, en ambiente de Html
<script>
    var a, b, c
    a = parseInt(prompt("Ingrese el primer número: "))
    b = parseInt(prompt("Ingrese el segundo número: "))
    c = a + b
    document.write("El valor de c = "+c)
</script>

En el lenguaje que estamos presentando: Lenguaje LCON:
Definir {
Var a,b: Entero;
}
Leer "El primer número: ",a;
Leer "El segundo número: ",b;
c = a+b;
Imprimir "La suma es: ",c;
